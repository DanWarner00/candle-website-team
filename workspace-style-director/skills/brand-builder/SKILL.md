1. Read `{project_dir}/client-brief.md`, `{project_dir}/content/summary.md`, and `{project_dir}/design/visual-research.md`.
2. Define brand personality: 3-5 adjectives with one sentence each explaining visual translation.
3. Write mood paragraph: how the site should feel on first visit.
4. Define visual strategy: how this design differentiates from competitors.
5. Define photography direction: how to use existing photos or what stock style to seek.
6. Write to `{project_dir}/design/brand-direction.md`.
7. Announce completion.
