1. Read `{project_dir}/client-brief.md` and `{project_dir}/content/summary.md` (especially Notes for Style Director).
2. Search the web to find 3-5 well-designed websites in the same or adjacent industries matching the brand tone.
3. For each: note color palette, typography, layout patterns, imagery use, whitespace, overall impression.
4. Search for current web design trends appropriate for this industry and audience.
5. Write to `{project_dir}/design/visual-research.md`: Reference Sites, Industry Visual Patterns, Trend Opportunities, Anti-Patterns (what to avoid).
6. Announce completion.
